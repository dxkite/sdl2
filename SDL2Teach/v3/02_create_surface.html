本课的内容牵涉到数据的结构问题，先给大家讲解一下在颜色是怎么表示的，学美术的大概听过混合色这个词吧？电脑上的颜色都是混合色，由基本的红、绿、蓝三种颜色混合而成，即RGB颜色，有木有人问透明度用什么表示？答案是alpha，由此，颜色的构成便有4个元素：红R、绿G、蓝B、透明度A；这是目前我遇到的比较完整的颜色表示，每一个元素取值0~255即一个字节，一个RGBA颜色占空间=sizeof(char)*8*4=32位，那么我们来建一个2x2的像素点内存

const int bitmap_w=2,bitmap_h=2;
//表面数据
Uint32 bitmap[]=
{
0xff00007f,
0xff0000ff,
0x00ff00ff,
0x0000ffff，
};

Uint32是SDL2中的一个类型，它是unsigned int类型的别名，一个Uint32数据占32位的储存空间，足以储存一个RGBA的颜色，我们来列表是、初始化一下，这个点阵含2x2=4个像素，分别为： 0xff00007f, 0xff0000ff, 0x00ff00ff, 0x0000ffff，我把这4种颜色用网页渲染出来了，从网页效果可以看出哪一部分数据代表那种颜色，如果还不明白，看看下面的图片：
